📧 Email Campaign Management System

📌 Description
A Python-based Email Campaign System that allows sending bulk emails using HTML templates.  
It simulates real-world email workflows including SMTP communication, basic deliverability handling, and campaign tracking.


🚀 Features
- Send bulk emails using CSV file
- HTML email templates with inline CSS
- Personalization (dynamic user names)
- Logging system (success & failure tracking)
- Retry mechanism for failed emails
- Simulated Mail Transfer Agent (MTA) workflow


🛠️ Technologies Used
- Python
- SMTP Protocol
- HTML (Email Templates)
- CSV (Data Handling)


📂 Project Structure
